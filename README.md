# AF-Booking-Case-Android-Multi-Module-Clean-Architecture-Mvvm-Compose-Kotlin-Hilt-Flow-Coil  
This project will demonstrate following:
- Material3
- Kotlin Programming Language
- Jetpack Compose
- MVVM
- Multi-Module with Clean Architecture
- Kotlin features - Coroutine, Scope Functions, Extensions, Flow, DI(Hilt)
